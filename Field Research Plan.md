您会看药物说明吗？3. 您有过忘记吃药、吃错药或者吃过期药的经历吗？若有，则经历是怎么样的？4. 您用什么方法来提醒自己吃药或防止自己吃错药呢？5. 您是否有购买到假药的经历？6. 您有手机吗？用的频繁不频繁？7. 您平时会不会用手机帮助你提醒自己吃药？如果没有，您觉得如果手机增加提醒吃药以及鉴别假药的功能您会使用吗？###Means of Introducing Core Questions:+ 首先和老人套近乎。（可以用上海话称呼其为“老先生”/“阿婆”）+ 您好，我们是上海科技大学的学生[1]，正在做社会实践调查，就是那种学校组织的活动[2]，想问您几个问题。+ 请问您今年高寿呀？+ 你看起来挺精神的嘛，一点都不像XX岁了。[3]+ 假设老人谦虚地表示自己身体某些部分感到不适，然后我们借机表示那您平时吃药吗？[4]然后如果老人回答了吃药，我们可以继续。如果回答不吃，那就记录为不吃药，然后寒碜几句下一位。+ 假设老人表示他平时都有锻炼，我们可以顺势问：身体这么好，那您平时一定不怎么吃药吧。如果他说他平时不吃药，我们可以记录为不吃药，然后寒碜几句下一位。如果回答还是吃的，那就继续。+ 然后表示：现在药上的字印得真小，看得人眼晕，您看得清药盒那些字吗？（考虑到老人可能吃中药的情况，也可以问：这些袋子长得都很像啊，您是怎么分清它们的呢？）您吃药的时候遇到过忘吃药或吃错药的情况吗？没有的话就问：您真厉害，那您有什么窍门避免吃错药或者忘吃药吗？+ 那您上一次核对药物的保质期是什么时候？+ 您以前有没有买到过假药呢？如果他回答有，我们继续问：具体是什么情况呢？+ 之后说我们准备为手机增加提醒吃药以及鉴别假药的功能，需要问几个关于手机使用情况的问题。然后问问题6、7.###Annotation: + [1]点明是哪个大学，即使老人不清楚，也能降低其防备心理。如果对方说是不是并入上大的那个上海科学技术大学，要强调是新成立的+ [2]表明来意，解释一下说是学校活动，老人会比较能理解。借由老人们对小辈关爱的心理，进一步降低防备，方便打开话题，继续问题。+ research的时候，我们分析了我们各自家中老人的情况，发现电子设备对老人的生活有很大的影响。在之后的讨论中，由于我们认为手机对老人的幸福程度的影响是不确定的——社交及网络功能能帮老人排遣孤独，而频繁的诈骗和推销电话影响到了老人的正常生活，因此我们将最初课题中的“电子设备”限定为手机，试图在之后的调查中了解手机对老人生活的总体影响究竟是正是负，并根据结果推广或限制老年人使用手机。但是在进行初步desktop research的时候我们发现了很多问题：首先手机在老年人群中的使用极为普遍，全球有90%的老年人已经在使用手机，因此老年人使用手机已经是一个必然的趋势，若想阻止老年人使用手机，成功的可行性是很小的。其次，手机的功能覆盖面极广，并且将来很可能会有很多新的实用的功能出现，因此让老人放弃使用手机是不可取的。于是我们开始研究如何利用手机使老年人更幸福。+ 我们初步设想可以利用手机改善老人的健康状况，因此我们在mind map上寻找与科技及健康相关的点，并进行了讨论。在讨论中我们发现家中的老人基本上都需要吃药，并且有些需要吃很多种药。而老人在吃药时常常犯错，如因忘记吃过药而多吃药、看不清药名服错药物、忘记按时吃药、不注意保质期吃过期药等等问题。我们认为这是一个待解决的问题，老年人存在在用药方面需要提醒的需求，而提醒功能可以借手机来实现。因此我们最终决定将课题方向定为处理老年人用药问题。+ 我们对于老年人的用药问题进行了desktop research，发现有59.5%[1]老人患有慢性病，需要服药。因为老人常常存在视力问题，并且有一部分文化程度不高的老人存在，而这样的老人在获取药物相关信息方面存在较大困难，因此我们假设在需要服药的老年人群中会发生吃错药、忘吃药、吃过期药、重复吃药的情况。事实上，通过desktop research，我们发现用药错误导致的估计死亡人数仅在美国一国就达到了125000人[2]。根据这些数据我们进一步假设，老人用药错误会对老人的健康有较大负面影响，从而使老人的幸福程度降低。+ 在desktop research之后，我们结合desktop research的结果以及个人经验提出了如上的assumptions。其中1、10、12是由desktop research提出的，而其余assumptions则是根据个人经验提出的。+ 根据我们提出的assumptions，我们制作了一份interview script。问题1可以检验assumption 1，问题2可以检验assumption 2、3 ，问题3可以检验assumption 4-9，问题4可以检验assumption 10，问题5可以检验assumption 11，问题6可以检验assumption 14，问题7可以检验assumption 15。assumption 12由desktop research可以证明[2]，而assumption 13是基本的假设，已被科学界广泛接受。从而，我们设计的interview script结合desktop research 可以检验我们之前提出的所有assumptions。+ 考虑到若我们直接询问我们的core questions则老人不一定会如实回答，使得interview的结果与现实状况有较大出入，因此我们设计了Means of Introducing Core Questions部分，试图使老人向我们敞开心扉，尽可能减少老人的回答与他们的真实情况之间的偏差，并且同时给予老人关怀。+ 在分工方面我们试图将每个人的特长发挥出来，使小组效率尽可能的最大化。本小组中有两位会说上海话的同学（刘孟加薇、王立翱），其中刘孟加薇同学上海话水平较好。为了降低产生语言障碍的可能性，刘孟加薇同学被分到了需要更多交流的养老院组（interview），而王立翱同学被分到了曙光医院组（observation+interview）。在剩余的三位同学中，只有储张恺文同学能听懂上海话，因此被分到了养老院组。考虑到英语水平问题，曲原辰同学被分到了曙光医院组，胡岩鹏同学参加讲座。+
